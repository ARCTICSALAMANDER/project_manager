# Менеджер проектов для программистов
# by ARCTICSALAMANDER
Десктопное приложение на PyQt6 для управления проектами, специально разработанное для программистов. Этот инструмент помогает организовать рабочий процесс, отслеживать задачи и управлять идеями проектов через интуитивно понятный графический интерфейс.
# Возможности
## Главное окно (MainWindow)
1. Отображает все проекты в прокручиваемом списке
2. Кнопка "Добавить проект" для создания новых проектов
3. Чистый интерфейс в темной теме, оптимизированный для разработчиков
## Управление проектами (ProjectLabel)
1. Умное именование: Уникальные имена проектов с валидацией
2. Авто-именование: Обрабатывает случаи, когда пользователь закрывает диалог выбора имени
3. Отслеживание статуса: Автоматический расчет статуса на основе выполнения задач:
3.1 Запланирован (0% выполнено)
3.2 Начат (1-99% выполнено)
3.3 Закончен (100% выполнено)
4. Выгрузка информации о задачах конкретного проекта в текстовый файл по одному клику
5.  Быстрые действия: Переход к проекту или удаление в один клик
## Рабочее пространство проекта (ProjectWindow)
1. Карта мыслей: Визуальная организация идей с иерархической структурой
2. Управление задачами: Список задач с отслеживанием выполнения
3. Интеграция консоли: Встроенная командная консоль для операций с проектом
4. Привязка папок: Связь с физическими директориями проектов
5. Навигация: Легкое переключение между проектом и главным окном
## Система задач (Task)
1. Гибкие задачи: Стандартные и пользовательские задачи
2. Стандартные задачи (автоматически создаются для каждого проекта):
2.1 "Написать ТЗ"
2.2 "Создать Git-репозиторий"
2.3 "Сделать первый коммит"
3. Пользовательские задачи: Определяемые пользователем с полным набором операций
4. Управление сроками: Выбор дедлайнов через календарь
5. Отслеживание выполнения: Отметка выполнения через чекбоксы
## Консольные команды (Console)
1. “help”: Показывает доступные команды и синтаксис
2. “show_statistics”: Аналитика проекта включая:
2.1 Последнюю выполненную задачу
2.2 Ближайшие дедлайны
2.3 Процент выполнения
2.4 Статистику задач
3. “bind_folder <путь>”: Привязка проекта к директории с интеграцией Git
## Интеграция с Git (GitManager)
1. Автоматическое обнаружение: Находит Git в системном PATH
2. Проверка репозитория: Проверяет наличие Git-репозитория
3. Валидация коммитов: Проверяет наличие коммитов
4. Бесперейная интеграция: Автоматическое обновление статуса для Git-задач
## Карта мыслей (IdeaMap & Idea)
1. Визуальная организация: Расстановка идей перетаскиванием
2. Иерархическая структура: Родительско-дочерние отношения между идеями
3. Интеллектуальное редактирование: Добавление, удаление и организация мыслей
4. Защита корневой идеи: Предотвращает случайное удаление корневой идеи
5. Визуализация связей: Линии, показывающие отношения между идеями
## Сохранение данных (SQLController)
1. SQLite база данных: Надежное локальное хранение данных
2. Три основные таблицы:
2.1 ProjectsList: Метаданные проектов и привязанные папки
2.2 Tasks: Полное управление задачами с дедлайнами
2.3 Ideas: Иерархические данные карты мыслей с позиционированием
3. Автосохранение: Сохранение данных между сессиями
## Техническая архитектура
### Frontend
1. PyQt6: Современный GUI-фреймворк с темной темой
2. QListWidget: Эффективное отображение списка проектов
3. QGraphicsView: Продвинутая визуализация карты мыслей
4. Кастомные виджеты: Специальные UI-компоненты для оптимального UX
### Backend
1. SQLite: Легковесная файловая база данных
2. Subprocess/Shutil: Системные операции с Git
3. Объектно-ориентированный дизайн: Модульный, поддерживаемый код
4. Событийно-ориентированная архитектура: Отзывчивые пользовательские взаимодействия
